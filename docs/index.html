Arquitetura em 30s

Como as peças se encaixam

Entrada segura

WireGuard

Túnel para 10.8.0.0/24 (exemplo), DNS interno e UFW restritivo liberando Guacamole/Vault/Grafana só para a VPN.

Credenciais efêmeras

Vault + Guacamole

AppRole gera usuários MySQL de curta duração; Tomcat/guacd recarregam sem downtime via timer.

Observabilidade

Loki + Promtail + Grafana

Coleta de syslog, WireGuard, Guacamole, NGINX e auditoria do Vault com dashboard pronto.

Antes de usar

Substitua os placeholders

Todos os segredos foram removidos. Preencha com valores reais antes de implantar.

Domínio

<ACCESSHUB_DOMAIN> → ex.: accesshub.suaempresa.com

VPN Pública

<VPN_PUBLIC_IP> → ex.: 203.0.113.10:51820

Gateway

<VPN_GATEWAY_IP> → ex.: 10.8.0.1

WireGuard

<WG_SERVER_PUBLIC_KEY> → chave pública do servidor.

Credenciais padrão

Altere usuários/senhas de exemplo (ex.: admin) conforme a política de segurança.

Entrega

Checklist rápido

  • Placeholders trocados e chaves/leases gerados novamente.
  • Ranges de rede e portas revisados no firewall.
  • Certificados e redirecionamentos HTTPS validados.
  • Dashboard de observabilidade atualizado e datasources ok.
  • Planilha de evidências do TESTES.md preenchida.
